# Project Manifest: Save & Grow

"가족 고정비를 찾아내고, 원클릭으로 줄여, 자동으로 투자하는 AI 머니 레이더"

## 🎯 우리의 미션
가족의 무의식적인 지출을 찾아내어, 가장 쉬운 방법으로 줄이고, 그 돈이 자산이 되는 완결된 경험을 제공한다.

## 💡 핵심 가치 (Core Values)

1. **Zero-Friction (극도의 간편함) - *New***
   * 사용자의 '귀찮음'이 우리의 적이다. 진단은 1분 안에 끝나야 하고, 해지와 절감은 원클릭(혹은 최소한의 뎁스)으로 이루어져야 한다.
2. **Transparency (투명성)**
   * 가족의 모든 고정비를 한 화면에서 볼 수 있어야 하며, 숨겨진 지출이나 이해하기 어려운 금융 용어는 배제한다.
3. **Connectivity (연결성)**
   * "줄인다"에서 멈추지 않는다. 줄인 고정비 → 투자 재원 → 자산 성장의 선순환 구조를 끊김 없이 연결한다.
4. **Action-Oriented (행동 중심)**
   * 단순한 가계부 기록이 아닌 "지금 당장 해야 할 행동(Action Item)"을 제안한다. (예: "지금 SKT 고객센터에 전화해서 이 대본을 읽으세요.")
5. **Family-Centric (가족 중심)**
   * 개인의 소비를 통제하는 것이 아니라, 가족 공동의 목표를 향해 나아가는 동기부여와 연대감을 형성한다.

## 🚫 우리가 만들지 않는 것
* 로그인 전부터 과도한 데이터 연동을 요구하는 무거운 온보딩
* 구성원 전원의 동의와 설치가 강제되는 뻣뻣한 시스템
* 절약과 단절된, 수수료 따먹기 식의 단순 금융 상품 추천
