\# MANIFEST.md



\##  Project Name

Save \& Grow (AI Money Radar)



\##  One-line Definition

가족 고정비를 줄이고, 절감된 돈을 자동으로 투자까지 연결하는 AI 기반 재무 최적화 서비스



\---



\##  Our Mission

우리는 사람들이 “돈을 아끼는 것”에서 멈추지 않고,

“아낀 돈을 성장시키는 것”까지 자동으로 연결되는 시스템을 만든다.



\---



\##  Core Problem

사람들은 돈을 아낄 수는 있지만,

그 돈을 투자로 연결하지 못한다.



\- 고정비는 비효율적으로 유지됨

\- 절약은 하지만 투자로 이어지지 않음

\- 행동으로 이어지지 않는 추천은 의미 없음



\---



\##  Our Solution

우리는 단순한 금융 관리 앱이 아니라,

\*\*“절약 → 투자까지 자동 연결하는 실행 시스템”\*\*을 만든다.



\### 핵심 구조

1\. 가족 단위 비용 통합 분석

2\. AI 기반 절감 기회 탐지

3\. 행동 유도 (협상, 해지, 변경)

4\. 절감 금액 자동 투자 연결



\---



\##  Principles



\### 1. Action over Insight

단순한 분석이 아니라 \*\*행동을 유도하는 서비스\*\*



\### 2. Everyday Use

사용자가 매일 또는 매달 자연스럽게 사용하는 구조



\### 3. Automation First

가능한 한 자동화 (Auto-saving, Auto-investing)



\### 4. Transparency

사용자가 이해할 수 있는 방식으로 설명



\### 5. Family-centric Design

개인이 아닌 가족 단위 재무 최적화



\---



\##  Why AI?



AI는 단순 계산이 아니라:

\- 사용자 패턴 분석

\- 절감 가능성 탐지

\- 개인 맞춤 투자 전략 생성

\- 행동 유도 메시지 생성



&#x20;AI 없이는 개인화 + 자동화 불가능



\---



\##  Vision



“모든 가정이 낭비되는 돈 없이,

자동으로 자산이 성장하는 구조를 만든다.”



\---



\##  Our Direction



우리는 단순한 가계부가 아니라,

\*\*“AI 기반 자동 자산 증식 엔진”\*\*을 만든다.
